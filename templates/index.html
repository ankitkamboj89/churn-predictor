Customer Churn Predictor

This web app predicts whether a customer is likely to churn (leave the service) based on demographic, usage, and account information. It is built using a Random Forest Classifier, trained on the Telco Customer Churn dataset.

The pipeline involves preprocessing categorical features with label encoding, standardizing numeric columns, and training a model with an 80/20 train-test split.

Model Used: Random Forest Classifier
Evaluation Metric: F1-score
F1 Score (Yes churn): ~0.63     F1 Score (No churn): ~0.86
Overall Accuracy: ~79%
